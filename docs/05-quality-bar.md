# Barra de calidad (mínimo aceptable)

Un tema está "OK" cuando incluye:
- Definición precisa
- Por qué importa (impacto real)
- Cuándo usar / cuándo no usar (reglas de decisión)
- 2+ trade-offs reales
- 1 mini-ejemplo (si aplica)
- 3+ preguntas trampa con respuestas
- Enlaces a prerequisitos y temas relacionados

Un tema es "Excelente" cuando además incluye:
- Modos de fallo (cómo se rompe en producción)
- Anti-patrones concretos
- Señales/observabilidad (métricas/logs)
- "Alternativas y comparación"
